# Montaje de un NAS doméstico (Synology)

## Autor

Alexis
Sistemas Informáticos

## Descripción

Este proyecto consiste en el montaje y configuración de un NAS (Network Attached Storage) orientado a uso doméstico, priorizando una solución económica y funcional.

Se ha utilizado un NAS Synology configurado en RAID 1 para garantizar la seguridad de los datos, permitiendo el acceso desde múltiples dispositivos dentro de la red.
